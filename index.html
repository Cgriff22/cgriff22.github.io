What This Project Covers

🌐

DNS Resolution

The Domain Name System translates our human-readable domain into a machine-routable IP address. See live nslookup and dig output on the Networking page.
📡

IP Addressing

Every device on the internet is identified by an IP address, either IPv4 (32-bit) or IPv6 (128-bit). GitHub Pages resolves to multiple addresses for load distribution.
🔒

HTTPS / TLS

HTTPS wraps HTTP inside a TLS tunnel, encrypting all data in transit. GitHub Pages enforces HTTPS by default with certificate-based authentication.
📦

TCP/IP Stack

Data moves through layers: Application (HTTP), Transport (TCP), Internet (IP), and Network Access (Ethernet/Wi-Fi). Each layer encapsulates the one above it.
